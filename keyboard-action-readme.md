# Keyboard - Action

Action has 15 Units of float numbers.
- Unit 0 = Arrow UP/DOWN to move forward/backward
- Unit 1 = Arrow LEFT/RIGHT to move left/right
- Unit 2 = Key R/F to rotate the whole body of the robot
- Unit 7 = Key O/L to lift shoulder
- Unit 9 = Key U/J to elbow flex
- Unit 11 = Key Y/H to wrist flex
- Unit 12 = Key Number 1/2  for wrist roll
- Unit 13 = Key Number 3/4 for gripper
